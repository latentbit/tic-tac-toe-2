# tic-tac-toe-2

- How long has it been since I built my first Tic-Tac-Toe project? So nostalgic 😔

# Project Description
- In this project, I focused on creating a mechanism that determines all possible winning positions based on the player's last move, regardless of board size.
- I also made an effort to organize the code, apply encapsulation, and use as few global variables as possible.
